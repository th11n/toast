# Toast monorepo

Starter dla publikowalnego komponentu React oraz aplikacji Next.js do jego rozwijania.

## Wymagania

- [Bun](https://bun.sh) 1.3 lub nowszy

## Start

```sh
bun install
bun run dev
```

Demo będzie dostępne pod `http://localhost:3000`.

## Skrypty

```sh
bun run build
bun run lint
bun run typecheck
bun run test
```

Pakiet biblioteki znajduje się w `packages/toast`, a aplikacja demonstracyjna w `apps/demo`.
